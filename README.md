你不必非得看过多少本书，但你要看懂这沉默，这黑夜，它教会你思想而不单是看书。-史铁生

我曾说过，只要人类存在，我们对万事万物还渴望着表达的话，文学依然是最佳途径，不会消亡。-迟子建
